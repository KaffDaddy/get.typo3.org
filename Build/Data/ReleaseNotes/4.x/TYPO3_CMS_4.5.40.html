Release Notes for TYPO3 CMS 4.5.40

This document contains information about TYPO3 CMS 4.5.40 which was released on February 19th, 2015.

News

This release is a combined bug fix and security release.

Notes

Due to a critical security issue found in the TYPO3 CMS Core, there was a release of TYPO3 CMS 4.5.40.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2015-001/
