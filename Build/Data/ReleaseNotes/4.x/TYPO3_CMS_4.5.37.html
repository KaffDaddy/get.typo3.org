Release Notes for TYPO3 CMS 4.5.37

This document contains information about TYPO3 CMS 4.5.37 which was released on October 22nd, 2014.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 CMS Core, there was a combined release of TYPO3 CMS 4.5.37, 4.7.20, 6.1.12 and 6.2.6.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2014-002/
