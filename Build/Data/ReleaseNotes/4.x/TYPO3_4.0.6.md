Upgrading
---------

This version contains a fix that makes the TYPO3 Rich Text Editor
(rtehtmlarea) work again in Mozilla Firefox 2.0.0.3. To ensure that this
will work, you need to remove all \*.js files inside the typo3temp/
directory!
