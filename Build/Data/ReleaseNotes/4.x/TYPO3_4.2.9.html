Release Notes for TYPO3 4.2.9

This document contains information about TYPO3 version 4.2.9 which was released on September 28, 2009.

News

This release is a bugfix release.

Upgrading

The usual upgrading procedure applies, but a database update is necessary! Use the Install Tool, section “2: Database Analyser” to Compare the database and apply the changes proposed (if any).
