Release Notes for TYPO3 4.5.17

This document contains information about TYPO3 version 4.5.17 which was released on July 4th 2012.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 Core, there was a combined release of TYPO3 4.5.17, 4.6.10 and 4.7.2.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2012-003/
