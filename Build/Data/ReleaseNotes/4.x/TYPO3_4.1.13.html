Release Notes for TYPO3 4.1.13

This document contains information about TYPO3 version 4.1.13 which was released on October 22, 2009.

News

This release is a bugfix release.

Notes

Due to several security issues found in the TYPO3 Core, there was a combined release of TYPO3 4.1.13, 4.2.10 and 4.3.0beta2.
Find more details in the security bulletin: https://typo3.org/news/article/multiple-security-issues-found-in-typo3-core-1/
