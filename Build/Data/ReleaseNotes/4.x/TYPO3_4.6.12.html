Release Notes for TYPO3 4.6.12

This document contains information about TYPO3 version 4.6.12 which was released on August 15th 2012.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 Core, there was a combined release of TYPO3 4.5.19, 4.6.12 and 4.7.4.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2012-004/

Upgrading

The usual upgrading procedure applies. No database updates are necessary.

Changes

Here is a list of what was fixed since 4.6.11:

  • [RELEASE] Release of TYPO3 4.6.12
  • [SECURITY] XSS in install tool (#21634)
  • [SECURITY] Page Link Target vulnerable to XSS (#32653)
  • [SECURITY] XSS in validateForm (#25052)
  • [SECURITY] XSS in TCE forms (#25356)
  • [SECURITY] XSS in Scheduler Example Task (#30967)
  • [SECURITY] HTML5 support in RemoveXSS (#37127)
  • [SECURITY] Information Disclosure in the Configuration Module (#39345)
  • [SECURITY] Untrusted GP data is unserialized in old CSH handling (#33520)
  • [SECURITY] XSS in Indexed Search statistics (#31927)
  • [SECURITY] t3lib_div::quoteJSvalue allows XSS (#23226)
  • [TASK] Set TYPO3 version to 4.6.12-dev
