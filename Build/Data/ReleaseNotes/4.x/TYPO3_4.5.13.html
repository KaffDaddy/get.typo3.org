Release Notes for TYPO3 4.5.13

This document contains information about TYPO3 version 4.5.13 which was released on March 13th 2012.

News

This version is a maintenance release and contains bug fixes only.

Upgrading

The usual upgrading procedure applies. No database updates are necessary.
