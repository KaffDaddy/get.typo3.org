Release Notes for TYPO3 4.2.12

This document contains information about TYPO3 version 4.2.12 which was released on February 23, 2010.

News

This release is a combined bugfix and security release.

Notes

Due to several security issues found in the TYPO3 Core, there was a combined release of TYPO3 4.4-alpha1, 4.3.2 and 4.2.12.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-sa-2010-004/
