Release Notes for TYPO3 CMS 4.5.30

This document contains information about TYPO3 CMS 4.5.30 which was released on September 12th, 2013.

News

This release is a bug fix release.

Notes

This is part of a combined release of TYPO3 CMS 4.5.30, 4.7.15, 6.0.10 and 6.1.5.

Upgrading

The usual upgrading procedure applies. No database updates are necessary.

Changes

Here is a list of what was fixed since 4.5.29:

  • [RELEASE] Release of TYPO3 4.5.30
  • [BUGFIX] Backend Layout Grid Wizard not fully visible in Mac Firefox 22 (#50424)
  • [BUGFIX] Exclude empty passwords from password hashing check (#36244)
  • [BUGFIX] Escape title tag of image links (#50760)
  • [BUGFIX] Page tree filtering broken in IE7 & IE8 (#25327)
  • [BUGFIX] TCA: subtypes_addlist not processed (#30636)
  • [BUGFIX] Handle file_references properly in t3lib_befunc::thumbCode (#31295)
  • [BUGFIX] number_format() expects parameter 1 to be double (#50868)
  • [BUGFIX] MySQL: Use ENGINE (not TYPE) for storage-engine (#50466)
  • [BUGFIX] Syntax error in class.tx_coreupdates_t3skin.php (#35274)
  • [BUGFIX] Fix fatal error in extension manager (#46642)
  • [TASK] Provide information about import action in TCEmain to hooks (#36031)
  • [TASK] Set TYPO3 version to 4.5.30-dev
