Release Notes for TYPO3 4.7.14

This document contains information about TYPO3 CMS 4.7.14 which was released on July 30th 2013.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 Core, there was a combined release of TYPO3 CMS 4.5.29, 4.7.14, 6.0.8 and 6.1.3.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2013-002/
