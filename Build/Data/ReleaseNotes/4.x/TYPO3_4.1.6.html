RELEASE NOTES FOR TYPO3 4.1.6

This document contains information about TYPO3 version 4.1.6 which was released on March 3, 2008.

News

TYPO3 4.1.6 is entirely a bugfix release. A total of 21 issues has been resolved ranging from subjects like Inline Relational Record Editing (IRRE) over Indexed Search (Backend module DBAL compliance) to the CSV export of the list module. Also, the central command line invocation script cli_dispatch.phpsh which is used to start up things like the Indexed Search Crawler or the Clean Up Tools (tx_lowlevel_cleaner) has been fixed to work flawlessly on Windows systems now. Issues with ignoring extension constraints in the Extension Manager have also been resolved.

Thanks to the core developers and other community members involved in the bugfixing!

Download

https://typo3.org/download/

Upgrading

Nothing special to do, no database updates necessary (as usual, for patch level releases).
