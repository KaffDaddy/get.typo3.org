Release Notes for TYPO3 4.6.11

This document contains information about TYPO3 version 4.6.11 which was released on August 8th 2012.

News

This release is a bug fix release.

Upgrading

The usual upgrading procedure applies. No database updates are necessary.

Changes

Here is a list of what was fixed since 4.6.10:

  • [RELEASE] Release of TYPO3 4.6.11
  • [TASK] Raise submodule pointer
  • [BUGFIX] Exception “Could not create directory” (#39583)
  • [BUGFIX] Pass $fieldName when processing FlexForm DS in t3lib_transferData (#39527)
  • [BUGFIX] t3lib_db - expects parameter 1 to be resource, boolean given (#39509)
  • [BUGFIX] unlink issues warnings for lock files (#32282)
  • [BUGFIX] IRRE childs don't expand in Internet Explorer (#38849)
  • [BUGFIX] t3lib_db::exec_SELECTgetRows method annotation (#39417)
  • [BUGFIX] Properly check disabled versioning within tcemain (#33625)
  • [BUGFIX] PHP warnings may show up in the List module (#22152)
  • [BUGFIX] Shell command arguments are not escaped (#31278)
  • [BUGFIX] RTE transformation transforms LF/CR between div and hr into space (#26815)
  • [BUGFIX] Exclude E_STRICT from errors with PHP 5.4 (#35154)
  • [TASK] Always return a boolean in t3lib_div::validPathStr (#38604)
  • [BUGFIX] Invalid fallback for non-localized labels (#39220)
  • [TASK] Improve error message of “broken rootline” (#33082)
  • [BUGFIX] stdWrap numRows fails due to wrong SELECT clause (#34152)
  • [BUGFIX] Update extension must invalidate autoloader cache (#33895)
  • [TASK] Raise submodule pointer
  • [BUGFIX] QT movies prefixed abusively in Media CE (#39026)
  • [BUGFIX] Send sane HTTP response in showpic on error (#39052)
  • [BUGFIX] Remove bogus template in template analyzer (#38104)
  • [BUGFIX] Drag&Drop inside the root page of the pagetree isn't possible (#24626)
  • [BUGFIX] Check if user is allowed to paste page to pagetree (#33546)
  • [BUGFIX] Add rootline workspace overlay for backend_layouts. (#36313)
  • [BUGFIX] accessibilityWrap ignores simple value (#38791)
  • [BUG] t3lib_div::getFilesInDir order differs from order in Filelist (#18771)
  • [BUGFIX] RTE: Importing google webfonts breaks style sheet parsing (#36316)
  • [BUGFIX] RTE spellcheck issue on Windows server (#36438)
  • [BUGFIX] RTE 4.6: Force IE9 to use IE8 mode in frontend (#38658)
  • [TASK] Set TYPO3 version to 4.6.11-dev
