Upgrading

The usual upgrading procedure applies. No database updates are necessary.

Here is a list of what was fixed since 4.5.17:

  • [RELEASE] Release of TYPO3 4.5.18
  • [BUGFIX] Pass $fieldName when processing FlexForm DS in t3lib_transferData (#39527)
  • [BUGFIX] t3lib_db - expects parameter 1 to be resource, boolean given (#39509)
  • [BUGFIX] Suggest Wizard crashes in Frontend Editing (#25079)
  • [BUGFIX] unlink issues warnings for lock files (#32282)
  • [BUGFIX] IRRE childs don't expand in Internet Explorer (#38849)
  • [BUGFIX] t3lib_db::exec_SELECTgetRows method annotation (#39417)
  • [BUGFIX] Properly check disabled versioning within tcemain (#33625)
  • [BUGFIX] PHP warnings may show up in the List module (#22152)
  • [BUGFIX] Shell command arguments are not escaped (#31278)
  • [BUGFIX] RTE transformation transforms LF/CR between div and hr into space (#26815)
  • [BUGFIX] Exclude E_STRICT on PHP 5.4 and unify error reporting (#38691)
  • [TASK] Always return a boolean in t3lib_div::validPathStr (#38604)
  • [TASK] Improve error message of “broken rootline” (#33082)
  • [BUGFIX] Formmail doesn't always use correct character set (#28684)
  • [BUGFIX] $_EXTCONF was not filled in ext_tables.php (#38927)

  • [BUGFIX] E_DEPRECATED does not exist in PHP 5.2 (#38645)
  • [BUGFIX] RTE spellcheck issue on Windows server (#36438)
