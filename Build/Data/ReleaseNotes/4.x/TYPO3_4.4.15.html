Release Notes for TYPO3 4.4.15

This document contains information about TYPO3 version 4.4.15 which was released on April 17th 2012.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 Core, there was a combined release of TYPO3 4.4.15, 4.5.15 and 4.6.8.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2012-002/
