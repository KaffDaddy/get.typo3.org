Release Notes for TYPO3 4.5.10

This document contains information about TYPO3 version 4.5.10 which was released on December 20th, 2011.

News

This release is a maintenance releases and contains bugfixes only.

Notes

This release aims to fix one regressions that has been introduced with the provious release:

  • #32625: In the backend, custom stylesheets in the rich text editor have not been showing up anymore, due to a wrong file inclusion.
