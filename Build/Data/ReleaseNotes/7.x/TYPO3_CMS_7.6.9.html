Release Notes for TYPO3 CMS 7.6.9

This document contains information about TYPO3 CMS 7.6.9 which was released on May 24th, 2016.

News

This release is a regression fix release for TYPO3 CMS 7.6.8 which contained critical security fixes.

Upgrading

The usual upgrading procedure applies. No database updates are necessary.
It might be required to clear all caches; the “important actions” section in the TYPO3 Install Tool offers the accordant possibility to do so.
