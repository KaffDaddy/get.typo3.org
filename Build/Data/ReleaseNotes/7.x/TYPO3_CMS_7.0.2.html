Release Notes for TYPO3 CMS 7.0.2

This document contains information about TYPO3 CMS 7.0.2 which was released on December 10th, 2014.

News

This release is a combined bug fix and security release.

Notes

Due to security issues found in the TYPO3 CMS Core, there was a combined release of TYPO3 CMS 4.5.39, 6.2.9 and 7.0.2.
Find more details in the security bulletin: https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2014-003/

Upgrading

The usual upgrading procedure applies. No database updates are necessary.
It might be required to clear all caches; the “important actions” section in the TYPO3 Install Tool offers the accordant possibility to do so.
