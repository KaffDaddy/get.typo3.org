Release Notes for TYPO3 CMS 6.1.11

This document contains information about TYPO3 CMS 6.1.11 which was released on September 23rd, 2014.

News

This release is a bug fix release.

Notes

This is part of a combined release of TYPO3 CMS 4.5.36, 6.1.11 and 6.2.5.
