Release Notes for TYPO3 6.0.7

This document contains information about TYPO3 version 6.0.7 which was released on July 23rd 2013.

News

This release is a bug fix release.

Notes

This is a combined release of TYPO3 CMS 4.5.28, 4.7.13, 6.0.7 and 6.1.2.

The usual upgrading procedure applies. No database updates are necessary.
