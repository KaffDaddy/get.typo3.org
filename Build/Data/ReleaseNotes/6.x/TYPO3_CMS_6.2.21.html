Release Notes for TYPO3 CMS 6.2.21

This document contains information about TYPO3 CMS 6.2.21 which was released on April 12th, 2016.

News

This release is a regression fix release for TYPO3 CMS 6.2.20 which contained important security fixes.

Upgrading

The usual upgrading procedure applies. No database updates are necessary.
It might be required to clear all caches; the “important actions” section in the TYPO3 Install Tool offers the accordant possibility to do so.
