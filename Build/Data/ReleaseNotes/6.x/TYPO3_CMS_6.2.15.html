Release Notes for TYPO3 CMS 6.2.15

This document contains information about TYPO3 CMS 6.2.15 which was released on September 8th, 2015.

News

This release is a combined bug fix and security release.

Notes

Find more details in the security bulletins:

  • https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2015-008/
  • https://typo3.org/teams/security/security-bulletins/typo3-core/typo3-core-sa-2015-009/

Upgrading

The usual upgrading procedure applies. No database updates are necessary.
It might be required to clear all caches; the “important actions” section in the TYPO3 Install Tool offers the accordant possibility to do so.
