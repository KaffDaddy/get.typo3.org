Release Notes for TYPO3 CMS 6.2.10

This document contains information about TYPO3 CMS 6.2.10 which was released on February 26th, 2015.

News

This release is a bug fix release which also contains performance improvements.

Notes

Download

https://typo3.org/download/ or alternatively http://get.typo3.org/

MD5 checksums

97437e953b15b8445166c4dda615bda0  typo3_src-6.2.10.tar.gz
2df3f8a26daff609e3797a747baa5a52  typo3_src-6.2.10.zip

Upgrading

The usual upgrading procedure applies. A database update is not strictly necessary, but recommended in order to benefit from the bugfixes for 57584 and 64616.
